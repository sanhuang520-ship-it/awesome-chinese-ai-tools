## 🔥 今日重点

**💸 Claude Fable 5 今天起正式收费**

从今天（6月23日）开始，Claude Fable 5 已从 Pro / Max / Team / 企业版套餐额度中移除，继续使用需要消耗预付费 usage credits，按 API 价格计费：输入 **$10/M tokens**、输出 **$50/M tokens**——是 Anthropic 目前最贵的常规模型，价格为 Opus 4.8 的两倍。如果你日常需求用 Opus 4.8 已经够用，不必为 Fable 5 额外付费；只有在复杂推理、长周期 Agent 任务上才值得动用 credits。Anthropic 称未来产能充足后会考虑重新纳入套餐，暂无时间表。

**🚀 MiniMax M3 发布：稀疏注意力大幅降本**

MiniMax 发布新一代通用模型 **M3**，428B 参数（激活 23B），采用全新 **MSA 稀疏注意力**架构，支持 100 万 token 超长上下文，原生支持文本/图像/视频多模态。相比 3 月的 M2.7，百万 token 长度下预填充提速约 9.7 倍、解码提速约 15.6 倍，长上下文推理成本降至上一代的约 1/20。官方称这是国内首个同时集成编程、智能体协作与 Computer Use 能力的开源模型。
