## 🔥 今日重点

**🐋 DeepSeek V4.1 正式发布（6月22日发布，本周持续放量）**

DeepSeek V4.1 已于本周一（6月22日）正式发布，国产大模型迈入多模态时代。核心升级：

- **三模态原生支持** — 文本/图像/音频统一处理，音频延迟降低 8 倍
- **2M token 超长上下文** — 环形注意力 + 块预填充，推理速度 180 tokens/s
- **MCP 协议深度适配** — 智能体工具调用更顺畅
- **API 价格降至 ¥1/M 输入 tokens** — 降幅 75%，进一步拉低国产大模型使用门槛

本周已大范围推送给企业客户，个人用户可直接在 [chat.deepseek.com](https://chat.deepseek.com) 体验。

**⚠️ Claude Mythos 5 / Fable 5 对 Glasswing 合作伙伴临时不可用**

Anthropic 此前（6月9日）发布的 Mythos 5 和 Fable 5 模型，原计划通过 Project Glasswing 向 AWS、Apple、Google、JPMorgan 等核心安全合作伙伴提供。近日有报告显示该访问临时不可用，可能与基础设施容量管理相关。普通用户的 API 不受影响。如果你的工作流依赖企业级 Claude，需要留意。

## 🆕 今日推荐工具

**Windsurf** — AI 原生 IDE → [windsurf.com](https://windsurf.com)

由 Codeium 团队推出，与 Cursor 并列「AI 原生编辑器」第一梯队。核心特色是 **Cascade Agent 模式**：你只需描述任务，Agent 会自动跨多个文件理解上下文、修改代码、运行命令、查看结果。学生免费、个人免费版每月 50 次 prompt credits 够轻度试用，Pro 版 $15/月。国内访问需梯子。

> ⚡ 与 Cursor 的取舍：Cursor 更成熟、社区生态广；Windsurf 的 Agent 自主性更强、UI 更新潮。两个都值得装来对比。
