## 🔥 今日重点

**🎨 Google 上线 Gemini 3.1 Flash-Lite Image（6月30日）**

Google 于本周二发布 **Gemini 3.1 Flash-Lite Image**（社区昵称「Nano Banana 2 Lite」），主打**高速、低成本**的批量出图场景。已通过 Google AI Studio、Gemini API 和 Netlify AI Gateway 上线。这是对此前 `gemini-3.1-flash-image-preview` 预览版（已于 6/25 停用）的正式继任，模型支持文本/图像/音频/视频多模态输入。如果你的工作流用到批量出图（比如电商详情图、社媒配图），值得关注其定价是否比 Midjourney/即梦更划算。

**🚀 GPT-5.6 新增 Max 推理模式 + Ultra 多智能体模式**

OpenAI 为 GPT-5.6 Sol 追加两个新能力：
- **Max reasoning effort** — 给模型更长思考时间处理高难度任务
- **Ultra mode** — 突破单 Agent 限制，用多个子智能体（subagent）协同加速复杂工作

目前仍处于受限预览阶段，仅对精选合作伙伴通过 API 和 Codex 开放，OpenAI 表示会"尽快"扩大范围。这两个新模式方向上和 Anthropic 的「Effort Control」、Claude 的多智能体编排比较接近，说明「可调节推理深度 + 多智能体协作」正成为下一代旗舰模型的标配。
