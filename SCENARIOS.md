# 不同场景下，我会选哪个 AI 工具

> 工具收集容易，选对工具难。这里不堆链接，只说我实际用下来觉得值得推荐的。

---

## 🎓 我是学生，要写论文 / 做报告

**核心痛点：** 找资料慢、读英文文献费劲、写出来的东西干巴巴

| 场景 | 推荐工具 | 为什么 |
|------|----------|--------|
| 查资料、找文献 | [秘塔搜索](https://metaso.cn) | 无广告，支持"学术模式"，引用来源可追溯，比直接问 ChatGPT 靠谱 |
| 读英文 PDF | [沉浸式翻译](https://immersivetranslate.com) | 直接在浏览器里双语对照，不破坏原文排版，支持 PDF 和网页 |
| 润色中文表达 | [秘塔写作猫](https://xiezuocat.com) | 专门针对中文语法和学术表达，改出来的句子不像 AI 腔 |
| 长文档总结 | [Kimi](https://kimi.moonshot.cn) | 200k 上下文，直接扔进去一篇论文问重点，免费用 |
| 做 PPT | [Gamma](https://gamma.app) | 给它一段文字大纲，自动生成结构清晰的 PPT，模板比 WPS 好看 |

**一句话组合：** 秘塔搜索找资料 → Kimi 读文献 → 写作猫改稿 → Gamma 出 PPT

## 💻 我是程序员 / 开发者

**核心痛点：** 代码补全要快、问问题要准、不想老是切窗口

| 场景 | 推荐工具 | 为什么 |
|------|----------|--------|
| 日常代码补全 | [通义灵码](https://lingma.aliyun.com) | VS Code / JetBrains 插件，完全免费，国内访问不卡顿 |
| 复杂功能开发 | [Cursor](https://cursor.sh) | 真正理解整个项目上下文，不只是补单行，每月有免费额度 |
| 快速搭全栈原型 | [Bolt](https://bolt.new) | 说一句话出一个可运行的 web 应用，适合验证想法 |
| 技术问题搜索 | [Perplexity](https://perplexity.ai) | 比 Google 快，比 ChatGPT 准，引用 Stack Overflow 和文档原文 |
| 想省钱用 API | [DeepSeek](https://chat.deepseek.com) | 代码能力接近 GPT-4，API 价格是 OpenAI 的十分之一，性价比最高 |

**一句话组合：** 通义灵码日常补全 + Cursor 攻坚复杂任务 + DeepSeek API 做项目

## ✍️ 我是内容创作者（公众号 / 小红书 / 视频）

**核心痛点：** 选题没灵感、写了半天没人看、配图费时间

| 场景 | 推荐工具 | 为什么 |
|------|----------|--------|
| 写小红书 / 公众号 | [豆包](https://www.doubao.com) | 字节自家产品，对短内容平台的风格理解最准，免费 |
| 生成配图 | [即梦 AI](https://jimeng.jianying.com) | 中文描述直接出图，不需要学提示词，风格多样，每天有免费额度 |
| 视频脚本 → 视频 | [即梦视频](https://jimeng.jianying.com) | 图像和视频在同一个平台，工作流顺畅 |
| 翻译外网内容 | [沉浸式翻译](https://immersivetranslate.com) | 看英文资讯时双语对照，搬运选题效率翻倍 |
| 做封面图 | [文心一格](https://yige.baidu.com) | 国风、插画风格尤其好，每天免费电量够用 |

**一句话组合：** 豆包写文案 + 即梦出配图 + 沉浸式翻译找素材

## 🏢 我是职场打工人（汇报 / 会议 / 日常沟通）

**核心痛点：** 写周报烦、开会记录难、文档太多看不完

| 场景 | 推荐工具 | 为什么 |
|------|----------|--------|
| 写周报 / 总结 | [WPS AI](https://ai.wps.cn) | 直接在 WPS 里用，不用换工具，对中文职场文档格式最熟悉 |
| 长文档速读 | [Kimi](https://kimi.moonshot.cn) | 把合同、方案、报告直接丢进去问，省去通读时间 |
| 开会纪要整理 | [讯飞星火](https://xinghuo.xfyun.cn) | 讯飞的语音识别是国内最准的，录音转文字再整理成纪要 |
| 做数据分析报告 | [通义千问](https://tongyi.aliyun.com) | 支持上传 Excel，直接问数据规律，出图表 |
| 英文邮件 | [DeepSeek](https://chat.deepseek.com) | 写商务英文邮件措辞地道，免费，不限次数 |

**一句话组合：** Kimi 速读文档 + WPS AI 写材料 + 讯飞整理会议纪要

## 🖥️ 我想本地部署，数据不出本机

**核心痛点：** 担心数据隐私、想离线用、或者就是喜欢折腾

| 场景 | 推荐工具 | 为什么 |
|------|----------|--------|
| 本地跑大模型 | [Ollama](https://ollama.com) | 一行命令拉起模型，Mac / Linux 都支持，最简单的本地方案 |
| 本地知识库问答 | [FastGPT](https://fastgpt.in) | 开源，支持私有部署，把公司内部文档变成可问答的知识库 |
| 本地 LLMOps 平台 | [Dify](https://dify.ai) | Docker 一键部署，可视化编排 Agent，国产开源做得最完善 |
| 本地图像生成 | [Stable Diffusion WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui) | 跑起来之后无限出图，一次性成本，长期最划算 |
| 管理本地模型 | [LM Studio](https://lmstudio.ai) | 图形界面，选模型下载即用，适合不想敲命令的人 |

**一句话组合：** Ollama 跑模型 + Dify 编排应用 + FastGPT 搭知识库
