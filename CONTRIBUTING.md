# 贡献指南

感谢你愿意为 **Awesome Chinese AI Tools** 做贡献！

## 如何添加工具

1. Fork 本仓库
2. 在对应分类下按字母/拼音顺序添加一行
3. 格式如下：

```
| [工具名称](链接) 🆓/💰 🇨🇳/🌐 | 免费额度说明 | ★★★☆☆ | 一句话特点 |
```

4. 提交 Pull Request，标题格式：`Add: 工具名称`

## 收录标准

- **真实可用**：工具必须正常运营，非 Demo 或停更产品
- **AI 相关**：核心功能需基于 AI/ML 技术
- **有免费选项**：优先收录有免费额度的工具（付费工具需注明）
- **中文可用**：界面或内容支持中文（国际工具需标注 🌐）

## 评分标准（中文支持）

| 评分 | 说明 |
|------|------|
| ★★★★★ | 中文母语级，专门为中文优化 |
| ★★★★☆ | 中文流畅，偶有不自然 |
| ★★★☆☆ | 中文可用，有明显局限 |
| ★★☆☆☆ | 中文较差，建议用英文 |
| ★☆☆☆☆ | 不支持中文 |

## 不收录的内容

- 已停止服务的工具
- 纯英文界面且无中文计划
- 仅有企业版无个人可用版本
- 明显的广告/推广链接

## 更新免费额度信息

AI 工具的免费额度变化频繁，如发现信息过期，请直接提 PR 更新，无需开 Issue。
